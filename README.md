# Pufferfork
A fork of the Pufferfish project, which aims to stay up-to-date with new minecraft releases. All credits go to the authors of the original project; see [the pufferfish repo](https://github.com/pufferfish-gg/pufferfish)

### For features refer to the readme in the [original project](https://github.com/pufferfish-gg/pufferfish/blob/ver/1.21/README.md)


## Downloads
You can find the latest releases and builds in the Github Releases page

## Building

Building a Paperclip JAR for distribution:

```bash
./gradlew applyAllPatches
./gradlew createMojmapPaperclipJar
```
